# ACT Matrix

A mobile-first tool for the ACT (Acceptance & Commitment Therapy) matrix exercise. Built for a 100-day ACT matrix drawing challenge hosted by KAIST Meditation Science Lab.

The problem: when logging the daily matrix on mobile (e.g. posting to a group chat), it's tempting to just write free-form text — which defeats the point of a *matrix*, i.e. visually mapping thoughts and actions across quadrants. This tool keeps the 4-quadrant structure intact and exports a clean 1200×1200 PNG ready for posting.

Quadrants: Away Actions / Toward Actions / Unwanted Inner Experiences / Values & Important People.

Stack: Vanilla HTML/JS, Canvas API, PWA. DD design system (Cloud Dancer palette, DM Sans + Cormorant Garamond).
