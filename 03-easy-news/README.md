# easy-news (쉬운뉴스)

Paste a Japanese news article URL or text, and it gets rewritten at JLPT N3 level with furigana (ruby text) annotations. Powered by Claude API (Sonnet). Includes Japanese text-to-speech at 0.85× speed.

Requires a user-provided Anthropic API key (stored in localStorage at runtime).

Stack: Vanilla HTML/JS, Claude API, PWA with service worker.
