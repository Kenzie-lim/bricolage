# chaekgado (책가도)

UI mockup for generating Instagram-ready images inspired by *chaekgado* — a traditional Korean genre of bookshelf still-life paintings. Users select book titles and an accent color; the app would generate an AI-composed image in that style.

Currently a frontend-only prototype. The generative AI backend is not yet connected (placeholder images from imgur).

Stack: Vanilla HTML/CSS/JS.
